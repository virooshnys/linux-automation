# Phase 1 – Linux & Bash Automation

## System Health Script
- Created a Bash script to log system health metrics
- Metrics collected:
  - Disk usage (with warning for ≥80%)
  - Memory usage
  - CPU load
  - Hostname
  - Timestamp
- Output written to a persistent log file in the home directory

## Key Commands Learned
- df -h
- free -h
- uptime
- chmod +x
- awk
- bash shebang (#!/bin/bash)
